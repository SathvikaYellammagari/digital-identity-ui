# 🌐 Digital Identity UI - Responsive Profile Card

## 📌 Project Overview
This project is a modern **Digital Identity UI (Profile Dashboard)** built using HTML and CSS.  
It showcases a personal profile with skills, stats, and responsive design using media queries.

---

## ✨ Features
- 👤 Profile section with image and details
- 📊 Stats cards (Projects, Followers, Success rate)
- 💡 Skills section (HTML, CSS, JavaScript, React)
- 🎨 Glassmorphism UI design
- 📱 Fully responsive layout (mobile friendly)
- ⚡ Hover effects on cards

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- Flexbox
- Media Queries
